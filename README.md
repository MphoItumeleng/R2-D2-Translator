# 🔊 R2-D2's Morse Code Translator

A fun and interactive Morse Code Translator that plays real "dot" and "dash" sounds — just like R2-D2!

---

## 🚀 Features

- 📝 Text to Morse code conversion
- 🔤 Morse code to text decoding
- 🔊 Audio playback of Morse code (dot & dash sounds)
- ⚠️ Input validation and error handling

---

## 🛠️ Requirements

- Python 3.10+
- `playsound==1.2.2`:
  ```
  pip install playsound==1.2.2
  ```
